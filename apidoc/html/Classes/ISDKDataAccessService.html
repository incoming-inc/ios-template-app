– setUserFilteringKeys:error:

Set content-filtering user target data. All keys passed in this dictionary will appear as a targeting option in the Incoming control panel, and can be used to target specific videos to specific users.

- (void)setUserFilteringKeys:(nonnull NSDictionary<NSString*,NSArray*> *)filterInfo error:(NSError *_Nullable *_Nullable)error

Parameters

filterInfo

a dictionary of key-values to be used for targeting the user. All keys of this dictionary should be NSString instances. All values should be NSArray. (any other type will be discarded, and the error parameter set).
error

will points to an NSError instance if an error occured

Discussion

Set content-filtering user target data. All keys passed in this dictionary will appear as a targeting option in the Incoming control panel, and can be used to target specific videos to specific users.

Example: the host app may want to pass on the user’s favourite team as: @{@“team”:@[@“swans”,@“rabbitoh”]}. All videos targeted to swans OR rabbitohs fans will be shown to the user
